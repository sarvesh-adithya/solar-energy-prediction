# 🌞 Solar Energy Production Prediction

## 📌 Problem Statement
The objective of this project is to predict the annual solar energy production (kWh) using real-world environmental and project-related data.

## 📊 Dataset
- 10,000+ records  
- Features include environmental and system parameters  
- Target: Estimated Annual PV Energy Production  

## ⚙️ Approach
- Data cleaning and preprocessing  
- Feature engineering  
- Model: XGBoost Regressor  

## 📈 Results
- R² Score ≈ 0.85+  
- Improved prediction accuracy  

## 🛠️ Tools
- Python  
- Pandas, NumPy  
- Scikit-learn  
- XGBoost  

## 🚀 Future Improvements
- Streamlit dashboard  
- Real-time data integration
